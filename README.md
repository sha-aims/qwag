# QWAG

**QWAG** is a Python package and accompanying datasets for computational
experiments in weighted algebraic geometry.

## Overview

The goal of this project is to provide:

- implementations of algorithms related to weighted algebraic geometry
- datasets used in computational experiments
- tools supporting research in arithmetic geometry and related areas

The repository contains both the Python implementation and the datasets
used in ongoing research projects.

## Repository Structure

qwag/        Python source code  
datasets/    datasets used in experiments  
examples/    example scripts demonstrating usage
